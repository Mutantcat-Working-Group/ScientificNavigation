### 一、功能简介
- 基础功能分为“工具索引”和“AI提示词”两个板块，功能均针对信息检索方向
- 其中“工具索引”中收录了许多常用的科学信息检索网站、学术工具、权威检查网站
- “AI提示词”板块中提供AI提示词增强功能（勾选式），可以快捷指定角色和规则，生成提示词
- 两方面的内容力求真实、有效、有实际意义，内容仍在逐步更新

### 二、在线使用
- 官方地址：https://pdfreader.mutantcat.org/
- 第一次访问时可能需要加载内容，需要稍微等待几秒，首次加载之后就可以直接秒开了
- 首页上方可以根据标签和名称进行分类和筛选，点击搜索按钮开始搜索内容
- AI提示词页面可以选择一个角色、多个规则，选择完成后下方将生产提示词，复制提示词后填写问题部分即可
